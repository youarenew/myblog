# MyBlog

---

### 介绍

​	使用nodejs+express+mongo搭建简约版个人博客，界面优雅，功能丰富，小巧迅速

### 安装部署
	1. 安装node.js、express、mongo、redis
    2. 启动mongodb、redis
    3. 安装依赖 npm install 
    4. 修改配置 config.js
    5. 启动项目 node app.js / npm start
    6. 浏览器访问 'http://localhost:3000'
    7. done!
